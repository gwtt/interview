### 什么是 Redis

> Redis（Remote Dictionary Server）是一个开源的、键值对型的数据存储系统。基于内存也可持久化的日志型数据库，被广泛用于数据库、缓存和消息中间件。并且支持多种类型的数据结构，用于应对各种不同场景。支持事务，持久化，LUA 脚本以及多种集群方案等。



### Redis优缺点

> 优点:
>
> - 基于内存操作，性能高，读写快
> - 支持高并发，支持并发读写
> - 支持主从模式，支持读写分离
> - 有丰富的数据类型和特性
> - 支持持久化操作，不丢失数据
>
> 缺点:
>
> - 容量收到物理内存限制，不能实现海量数据的高性能读写
> - 相对于关系型数据库，不支持复杂逻辑查询，存储结构比较简单
